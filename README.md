Spam Email Classifier

This project predicts whether a message is Spam or Not Spam (Ham) using a simple Machine Learning model.

Technologies

Python

Pandas

Scikit-learn

Requirements

Install the required libraries:

pandas
scikit-learn

Or install using pip:

pip install pandas scikit-learn
How to Run

Run the program:

python spam_classifier.py

Enter a message.

Example:

Enter a message: You won a free lottery
Result: spam
